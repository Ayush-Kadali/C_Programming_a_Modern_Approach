# Programming Project 01

## Question 

Write a program that uses printf to display the following picture on the screen 

```
       *
      *
     *
*   *
 * *
  *
```

## Solution


```c
#include <stdio.h>

int main (void)
{

    printf("        * \n");
    printf("       *  \n");
    printf("      *   \n");
    printf(" *   *    \n");
    printf("  * *     \n");
    printf("   *      \n");

    return 0;

}
```
